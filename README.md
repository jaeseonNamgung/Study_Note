Study_Note는 내가 오늘 하루 몰랐던 부분이나 중요한 부분을 기록하기 위한 공간입니다. 이미 알고 있던 내용도 같이 적다 보면 몰랐던 내용을 못 보고 지나 칠 수 있기 때문에 저는 몰랐던 부분이나 꼭 알아야 되는 부분만 정리해서 매일 공부하기 전에 정리한 기록을 숙지하는 목적으로 Study_Note를 만들었습니다. 또한 저는 처음 공부한 내용은 정리하지 않을 계회입니다. 처음 공부한 것들은 전체적으로 봐야 되는 부분이고 너무 얕은 지식이기 때문에 두 번째 공부하고 난 부분에서 부족한 부분만 따로 정리해서 Study_Note를 작성할 계획입니다.
