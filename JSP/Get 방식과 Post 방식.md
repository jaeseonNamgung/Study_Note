# Get 전송 방식
- ```URL(주소)```에 파라미터를 추가하여 요청하는 방식
- ```http://주소...?이름=값&이름=값&이름=값...```
- 길이제한이 있다.
- 어떠한 값이 전달되는 지 사용자가 볼 수 있다.
- ex)
```jsp
<form ... method="get">
```
- get 방식은 게시판 글 조회, 인터넷 기사 보기, 쇼핑물 상품 확인등에 많이 사용된다.

# POST 전송방식
- ```HTTP request```의 body(본문)에 파라미터를 추가하는 방식
- 길이제한이 없다.
- 어떠한 값이 전달되는지 사용자가 볼 수 없다.
- ex)
```jsp
<form... method="post">
```
- 로그인이나 회원가입 처럼 보안이 필요한 곳에 사용된다.
