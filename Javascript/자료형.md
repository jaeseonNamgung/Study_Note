## 1. 숫자형(Number Type)
- 숫자형은 정수 및 소수점 두개로 나누어진다. 
- 숫자형은 다른 언어에서 int와 double 등으로 표현된다. 
- 숫자형에는 일반적으로 숫자 이외 ```Infinity```, ```-Infinity```, ```NaN``` 같은 특수 숫자 값이 표함되어 있다. 
  - ```Infinity```:  무한대 
  - ```NaN``` : 계산 중에 에러가 발생했다는 것을 의미

```javascript
let age = 25;
let height = 175.5;
```


## 2. bigint
- bigint는 (253-1)(9007199254740991) 보다 크거나 -(253-1) 보다 작은 정수를 나타낼 때 사용한다.(일반 숫자형으로는 사용할 수 없다.)
- bigint는 숫자 맨 뒤에 n을 붙여 사용해야 한다.

```javascript
const bigInt = 1234567890123456789012345678901234567890n;
```

## 3. 문자형(String Type)

자바스크립트에서 문자형은 따옴표로 표현한다.

- 큰따옴표 : " "
- 작은따옴표 : ' '
- 역따옴표: **` `**

자바 스크립트에서 큰따옴표와 작은따옴표는 문자형을 표현할 때 사용된다. 둘의 차이는 없다. 역따옴표는 문자 안에 변수나 표현식을 ${}안에 넣어 특정한 값을 표현하고자 할 때 사용한다. 

```javascript
let str = "큰따옴표로 문자형 표현";
let str2 = "작은따옴표로 문자형 표현";

-----------------------------------------------------------------------

let name = "홍길동";

let str = `저는 ${name}입니다.`; // 결괏값 : 저는 홍길동입니다.

let result = `이 값의 결과는 ${5 + 2}입니다.` // 결괏값 : 이 값의 결과는 7입니다.
```


## 4. 볼린형(Boolean Type)

볼린형은 맞다 아니다라고 판단하고자 할 때 사용한다. 볼린형 결과는 아래 두 가지로 표현된다.

- true : 맞다.
- flase : 아니다.


```javascript
let isSuccess = true;
let istest = false;

--------------------------------------

// 볼린형은 비교 결과를 저장할 때도 사용된다.

let isGreater = 6 > 2;  // true
```

## 5. null 
null은 어느 자료형도 속하지 않는 값이다. 
자바스크립트에서 null은 존재하지 않는 값 또는 비어있는 값이라는 뜻을 지니다. 

```javascript
let result = null;
```

## 6. undefined 
undefined는 값을 할당하지 않는 상태라는 뜻이다. 개발자가 명시적으로 undefined로 값을 할당할 수 있지만 추천하지 않는다. 값을 할당하지 않을 때는 null을 사용하는 걸 추천한다.

```javascript
let result = undefined;
```
