# 배열 사용법
- ```push('원소')``` : 배열의 맨 마지막 위치에 원소를 추가
- ```concat(['원소','원소',...])``` : 배열의 맨 마지막 위치에 원소를 여러 개 추가
- ```unshift('원소')``` : 배열의 맨 앞에 위치에 원소를 추가, 반환값이 있다.
- ```splice(위치, 변경할 위치 갯수, 변경할 원소,...)``` : 배열의 특정 위치에 원소를 변경, (위치, 변경할 위치 갯수, 변경할 원소,...)
- ```shift()``` : 제일 앞에 있는 배열 원소를 하나 제거
-```pop()``` : 맨 뒤에 있는 배열 원소를 하나 제거
- ```sort()``` : 빠른 순으로 정렬
- ```reverse()``` :느린 순으로 정렬 

```javascript
        let li = ['a', 'b', 'c', 'd', 'e'];

        li.push('f');

        document.write(li + "<br/>");

        li =  li.concat(['g', 'h']);  // 배열의 원소를 여러 개 추가하는 기능 , concat 은 반환 값이 있다.

        document.write(li + "<br/>");

        li.unshift('z');  // 배열의 첫 번째 자리에 추가 할때 사용, 배열을 추가하면 기존에 있던 원소들은 하나씩 밀리게 된다. 

        document.write(li + "<br/>"); 

        li.splice(2, 1, 'i', 'j');  // 배열의 특정 위치에 원소를 변경

        document.write(li + "<br/>"); 

        //---------------------------------------------------------------------------------

        document.write("------------------------------------------------" + "<br/>");
        let lii = ['a', 'b', 'c', 'd', 'e'];

        lii.shift();  // 제일 앞에 있는 배열 원소를 하나 제거

        document.write(lii + "<br/>");

        lii.pop();  // 맨 뒤에 있는 배열 원소를 하나 제거

        document.write(lii + "<br/>");

        let arrSort = ['c', 'e', 'a', 'b', 'd'];

        arrSort.sort();  // 빠른 순으로 정렬

        document.write(arrSort + "<br/>");
        
        arrSort.reverse();  // 느린 순으로 정렬

        document.write(arrSort + "<br/>");
```
