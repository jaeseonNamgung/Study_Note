# 테이블 레이아웃
테이블 레이아웃은 단어 뜻에서 유  즉 테이블 레이아웃은 행과 여러 개의 열을 넣어 레이아웃을 구성한다. 테이블 레이아웃 안에는 TableRow라는 태그가 여러 개 들어가는데 이 태그는 한 행을 의미한다. TableRow안에는 여러 개 뷰가 들어가며 TableRow안에 들어간 뷰는 행이 된다.

### stretchColumns
stretchColumns는 가로 방향으로 여유 공간이 있다면 그 여유 공간까지 모두 채워서 칼럼을 설정할 수 있는 속성이다. 
칼럼의 갯수는 0 부터 시작 하면 ,(콤마)를 사용해 인덱스를 설정할 수 있다. 만약 칼럼이 "0" 이라면 첫 번째 칼럼이 여유 공간을 채워진다. 두 개의 칼럼이 여류 공간을 추가로 차지하도록 만들고 싶다면 "0,1"로 설정하면 된다.
