# 레퍼런스(reference)란?
레퍼런스(reference)란 참조라는 뜻을 가진다. 즉 레퍼런스는 그 메모리에 값을 바로 사용하는게 아니라 다른 메모리에 값을 참조해서 다른 메모리에 값을 사용한다.
