# 제네릭이란?
제네릭이란 클래스 내부에 데이터 타입을 외부에서 정의하는 것을 말한다. 제네릭은 **<T>** 꺽세를 사용하며 <>안에 내용을 넣으면된다.
  
  
### 제네릭 사용법
  
 
```java
  
 class GeneriClass<T> { // 제네릭 사용

}

public class GeneriTest {

	public static void main(String[] args) {

		GenericTest<Integer> g = new GenericTest<Integer>();  // 메인 메서드에서 제네릭 데이터 타입 지정
		
	  GenericTest<String> g1 = new GenericTest<String>();  // 메인 메서드에서 제네릭 데이터 타입 지정
	}

}

 
```

GeneriClass에서 <T>안에 T는 아무 값도 없는 상태이다 즉 어떠한 데이터도 아니다. T는 메인 메서드에서 정의해준 데이터 타입에서 데이터 형이 정해진다.
