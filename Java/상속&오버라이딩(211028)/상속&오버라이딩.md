# 상속
- ```상속``` : 부모가 보유하고 있는 재산을 자식이 물려받는 것

## 상속의 특징
- ```extens``` 키워드를 통해 자식 클래스가 부모 클래스를 상속받을 수 있다.
- 자식 클래스는 부모 클래스의 모든 속성과 기능을 사용할 수 있다.
- 부모 클래스는 자식 클래스의 속성과 기능에 접근할 수 없다.
- 자바는 단일 상속 관계이기 때문에 하나의 자식 클래스는 여러 명에 부모 클래스를 둘 수 없다.


# 메서드 오버라이딩 
- 메서드의 '재정의'의 의미를 가진다.
- 부모의 메서드를 자식이 가져와서 자식에 맞는 기능을 재정의 해서 사용할 수 있다.

# super 
- ```super```는 부모 클래스를 가리킬 때 사용한다.
- 자식 클래스에 기본 생성자를 생성하고 사용할 경우 부모 클래스에 기본 생성자가 숨겨진 ```super``` 키워드로 인해 먼저 실행되고 자식 클래스에 생성자가 실행된다.
