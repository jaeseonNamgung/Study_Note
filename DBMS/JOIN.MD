# JOIN
- 두 개 이상의 테이블을 서로 결합하여 하나의 결과 집합을 만들어 낸다.
- 하나의 SQL 문을 통해서 여러 테이블의 데이터를 합쳐서 한 번에 조회할 수 있다.
- 주로 기본 키와 외래 키의 관계를 맺고 있는 테이블에 사용된다.
- 테이블의 결합 방식에 따라 **내부(INNER), 외부(OUTER), 자연(CROSS), SELF** 조인이 있다.
- 조인(JOIN)의 기본 형식
```SQL
SELECT 테이블1.속성, 테이블2.속성
FROM 테이블1 [조인 방식]JOIN 테이블2
ON 조인될 조건
WHERE 검색 조건; -- 선택 사항
```
