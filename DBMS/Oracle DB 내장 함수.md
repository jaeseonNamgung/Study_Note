# Oracle DB 내장 함수
## 문자 함수
- 문자 함수를 입력받아 문자나 숫자를 반환하는 함수 
1. 대소문자 변환 함수 : ```INITCAP(), LOWER(), UPPER()```
2. 문자열 길이 반환 함수 : ```LENGTH(), LENGTHB()```
3. 문자열 변경 함수 : ```CONCAT(), SUBSTR(), LPAD(), RPAD(), LTRIM(), RTRIM()```

- ```INITCAP('문자열')``` : 첫 글자는 대문자, 나머지는 소문자로 변환 EX) ```aPPLe -> Apple```
- ```LOWER('문자열')``` : 모두 소문자로 변환
- ```UPPER('문자열')``` : 모두 대문자로 변환

<br/>

- ```LENGTH('문자열')``` : 문자열의 글자 수 
-```LENGTHB('문자열')``` : 문자열의 바이트 수, 
  - 영문, 숫자 : 1byte
  - 한글 : 2byte 또는 3byte(utf-8)

<br/>

- ```CONCAT('문자열1', '문자열2')``` : 문자열1과 문자열2를 결합하는 함수
- ```SUBSTR('문자열', m, n)``` : 문자열에서 m 번째 문자부터 n 개의 문자열을 추출한다. 
  - 문자열에 시작 인덱스는 1이다.
  - m이 음수일 경우 문자열의 마지막부터 계산된다.
  - n을 생략할 경우 문자열 끝까지 계산된다.
- ```LPAD('문자열', m, '지정문자')``` : 문자열에 m 길이만큼 문자열 왼쪽 자리를 지정문자로 채워주는 함수
  - 지정문자를 생략할 경우 공백으로 값을 m 길이만큼 채운다.
- ```RPAD('문자열', m, '지정문자')``` : 문자열에 m 길이만큼 문자열 오른쪽 자리를 지정문자로 채워주는 함수
  - 지정문자를 생략할 경우 공백으로 값을 m 길이만큼 채운다.
- ```LTRIM('문자열', '지정문자')``` : 왼쪽에 있는 문자열에 지정문자를 모두 삭제하는 함수 
  - 지정문자를 생략할 경우 왼쪽에 있는 공백을 모두 삭제한다.
- ```RTRIM('문자열', '지정문자')``` : 오른쪽에 있는 문자열에 지정문자를 모두 삭제하는 함수 
  - 지정문자를 생략할 경우 오른쪽에 있는 공백을 모두 삭제한다.
- ```REPLACE('문자열', '변경할 문자열' , '변경될 문자열')``` : 특정 문자열로 변경하기 위한 함수

## 숫자 함수
- 숫자 데이터를 입력받아 처리(반올림, 버림, 나머지) 하는 함수
- ```ROUND(숫자, n)``` : 지정한 자릿수로 반올림하는 함수 
  - 소수점 n 자리로 반올림
  - n이 음수일 경우 정수부에서부터 n 자리로 반올림한다
- ```TRUNC(숫자, n)``` : 지정한 자릿수로 소수점을 버리는 함수
  - n을 지정하지 않을 경우 소수점을 전부 버린다.
  - n이 음수일 경우 정수부에서부터 n 자리로 전에 값을 버린다.
- ```MOD(숫자, n)``` : 나머지를 구하는 함수
- ```CEIL(숫자)``` : 소수점을 가장 가까운 정수로 올림
- ```FLOOR(숫자)``` : 소수점을 가장 가까운 정수로 내림

## 데이터 형태 변환 함수 
- 서로 다른 데이터 타입을 변환하는 함수 
1. ```TO_CHAR(숫자 또는 날짜)``` : 숫자 또는 날짜를 문자 형태로 변환
2. ```TO_NUMBER(문자)``` : 문자를 숫자 형태로 변환
3. ```TO_DATE(문자)``` : 문자를 날짜 형태로 변환

### 날짜 형식 모델
1. ```SYSDATE``` : 현재 날짜
2. ```YYYY OR YY``` : 연도
3. ```SCC: OR CC:``` : 세기(20세기 20세기)
4. ```MM``` : 월
5. ```DD``` : 일
6. ```MONTH OR MON``` : 월 
7. ```DAY``` : 요일
8. ```HH12: , HH24:``` : 시, ```MI:``` : 분, ```SS:``` : 초
